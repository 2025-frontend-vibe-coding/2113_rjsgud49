# 박건형 - Frontend Developer Portfolio

세련되고 다이나믹한 프론트엔드 개발자 포트폴리오 웹사이트입니다.

## 🚀 기술 스택

- **React 18** - UI 라이브러리
- **TypeScript** - 타입 안정성
- **Vite** - 빌드 도구
- **Framer Motion** - 애니메이션 라이브러리
- **React Icons** - 아이콘

## 📦 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 미리보기
npm run preview
```

## ✨ 주요 기능

- **스크롤 애니메이션**: 스크롤 시 섹션별 부드러운 애니메이션 효과
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기 지원
- **다이나믹 네비게이션**: 현재 섹션에 따른 활성화 표시
- **세련된 UI/UX**: 모던하고 깔끔한 디자인

## 📄 섹션 구성

1. **Hero** - 인트로 및 소개
2. **About** - 자기소개
3. **Skills** - 기술 스택 및 역량
4. **Projects** - 프로젝트 경험
5. **Profile** - 수상 경력 및 교육
6. **How I Work** - 업무 철학
7. **Contact** - 연락처 정보
